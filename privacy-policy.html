Privacy Policy — reel-crosspost

Last updated: July 3, 2026

reel-crosspost is a personal automation tool that publishes the operator's own video content to social media platforms (Instagram, Facebook, TikTok, YouTube).

What data this tool handles

  • OAuth access tokens issued by each platform to the operator's own accounts, used solely to authenticate API requests for publishing content. These tokens are stored locally on the operator's device and are never shared with, or accessible to, any third party.
  • Video files and captions provided directly by the operator for publishing. No data is collected from end users or viewers of the published content.

What this tool does not do

  • It does not collect, store, or process personal data belonging to any third party.
  • It does not use TikTok user data (or any platform's user data) for advertising, profiling, or any purpose other than publishing the operator's own content to the operator's own account.
  • It does not share any data with third parties or other services beyond the destination social media platform's own API.
